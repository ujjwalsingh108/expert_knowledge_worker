# Expert Knowledge Worker
It is an intelligent expert knowledge assistant that provides **curated**, **accurate** and **low-cost** answers to employees of insurance technology company named `Insurellm`.

A question answering assistant that is an expert knowledge worker. Employees of Insurellm uses it as an assistant that needs accurate and solutions to specific insurance related issues.

This project is based on `RAG (Retrieval Augmented Generation)` to ensure question/answering assistant has high accuracy.
